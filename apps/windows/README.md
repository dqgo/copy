# Windows 使用指南（用户版）

## 适用人群

希望在 Windows 设备上使用 Clipboard Sync 的普通用户。

## 安装

1. 从 GitHub Releases 下载 Windows 安装包（.exe）
2. 双击安装并启动应用

## 首次使用

1. 打开设置页面，确认工作空间 ID 与设备唯一 ID。
2. 选择同步通道：
	- 免费公共服务器（默认 kvdb.io 桶）
	- 私有 WebDAV
3. 执行连接测试。
4. 到配对页生成邀请码，发送给另一台设备；对端粘贴邀请码后发起配对。
5. 在状态页点击手动同步，或把同步模式切到 auto 开启共享剪贴板自动同步。

## 核心能力

- 主动配对：支持生成/解析邀请码，支持手动审批与自动审批策略。
- 免费公共服务器：内置公共中继模式（kvdb.io），可配置地址与共享桶 ID。
- 唯一 ID：首次启动自动生成并持久化设备唯一 ID，支持一键复制。
- 共享粘贴板：支持手动同步与自动轮询同步，出入站事件会写入历史记录。

## 常见问题

- 连接失败：检查网络与服务地址
- 配对后无同步：确认双方都在线，再手动同步一次
- 历史为空：先执行一次成功同步
