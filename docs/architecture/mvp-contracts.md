# 核心功能说明（用户版）

这份文档说明 Clipboard Sync 的基础能力边界，帮助你理解“能做什么、不能做什么”。

## 当前基础能力

- 读取本地剪贴板文本
- 将文本同步到已配对设备
- 将远端文本写回本地剪贴板
- 保存必要配置与信任设备

## 同步方式

- 手动触发同步（当前主路径）
- 部分平台支持前台快捷触发

## 设备管理

- 支持配对审批
- 支持拒绝配对
- 支持撤销已信任设备

## 平台差异说明

- iOS：受系统限制，后台持续监听能力有限
- Android：支持通知快捷动作与前台服务场景
- Windows/macOS：更接近桌面工作流交互
