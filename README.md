# Clipboard Sync

Clipboard Sync 是一款跨平台剪贴板同步工具，帮助你在多台设备之间更快地传递文本内容。

## 当前发布状态

| 平台 | 当前状态 | 安装形式 |
|------|----------|----------|
| Windows | 可使用 | 安装包（.exe） |
| Android | 可使用 | 安装包（.apk） |
| iOS | 准备中 | 即将提供 |
| macOS | 准备中 | 即将提供 |

说明：iOS 与 macOS 安装包正在整理中，当前仓库已包含对应功能说明与使用文档。

## 主要功能

- 设备配对与信任管理
- 手动同步文本剪贴板
- WebDAV 服务配置与连接测试
- 同步历史记录查看
- 多语言界面（中文/英文）
- 深色与浅色主题

## 快速开始（普通用户）

1. 打开 GitHub Releases，下载与你系统对应的安装包。
2. 安装并启动应用。
3. 在“设置”中选择服务模式：
   - 默认公共服务（开箱即用）
   - 私有 WebDAV（适合自托管）
4. 在两台设备上完成配对审批。
5. 点击“手动同步”验证收发是否成功。

## 隐私与安全

- 敏感配置使用系统安全存储（Keystore / Keychain / DPAPI）。
- 你可以随时撤销已信任设备并停止该设备同步。
- 提交问题时请不要公开你的账号密码和敏感剪贴板内容。
